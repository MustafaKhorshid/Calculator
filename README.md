# Calculator

A robust, interactive command-line calculator built using Python. This project serves as a foundational computer science portfolio piece, demonstrating clean coding practices, input validation, and numerical base conversions.

##  Features
- **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division using decimal values (`floats`).
- **Cybersecurity & Boundary Protection:** Implements strict exception handling to prevent application crashes from invalid user inputs, alongside explicit checks to prevent division-by-zero errors.
- **Base Systems Conversion:** Converts decimal integers into Binary (Base 2) and Hexadecimal (Base 16) formats using native string-slicing optimizations.

##  Technical Implementation
- **Language:** Python 3.x
- **Concepts Demonstrated:** Modular function design, infinite control loops (`while True`), conditional branching (`if/elif/else`), string manipulation, and input sanitization (`try/except`).
